Data-Cleaning-Course-Project
============================
This read.me is intended to provide the end-user with an understanding for my thought
process as I developed the run_analysis.R R program in support of the Getting and Data
Cleaning Course from Johns Hopkins through Coursera.

The effort was meant to summarize data into a tidy format from training and test data 
taken from the following study:
